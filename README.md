# v2ray-rulesets

Здесь собраны готовые пресеты правил для ПК и Android клиента, а также практически настроенный бекап ПК клиента

## Установка

- Скачать v2rayN
- Из релиза скачать бэкап
- Settings => Backup and Restore => Restore from local
- Settings => Routing Settings => RUv2 => Import Rules From Subscription URL
  - Выбираем Нет, чтобы заменить список
  - Жмём Confirm во всех окнах
- Help => Check Update => Check Update. Должны подгрузиться все geo файлы для xray/sing-box
- Добавляем свой профиль, выбираем как активный и радуемся

В дальнейшем новый список правил можно просто погружать из присоединённого URL

У конфигурации есть особенность: для YT+Google используется `byedpi` профиль на порте 1080. Надо отдельно скачать и запустить. Если не нужен, просто в роутинге надо заменить профиль на `proxy` вместо `byedpi`
